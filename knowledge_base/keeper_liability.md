# Source label: keeper_liability.md

Private parking correspondence may be addressed to the driver or the registered keeper. Notice to Keeper wording, timing, and required statutory information can affect how keeper liability is argued. Users should avoid accidentally admitting they were the driver if that fact is uncertain and relevant.

Useful materials include the original notice pages, envelope postmark or date of receipt if available, and a clear statement of what is and is not admitted. This area is technical; ParkPal only flags possible process points for user confirmation.
