# Source label: loading_unloading.md

Loading or unloading can be a factual defence where waiting was necessary to load goods or assist a passenger and site rules allow it, or where council restrictions distinguish loading from parking. Timing and activity matter.

Useful evidence includes delivery notes, collection receipts, photos of loading activity, and a minute-by-minute timeline. Distinguish genuine loading from waiting to shop or run an errand if the allegation is stopping in a restricted area.
