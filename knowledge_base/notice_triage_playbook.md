# Source label: notice_triage_playbook.md

Start with the document, not the user's assumption. Record the issuer, notice type, reference, vehicle registration, location, alleged event, notice date, service/received date, amount, discount wording, payment route, appeal route and every stated deadline. Flag any field that is missing, unclear or extracted with low confidence for manual confirmation.

Route triage: council PCN, private parking charge, Notice to Keeper, debt collection, court/legal document, possible scam, or unknown. Do not call a notice a scam merely because it looks unfamiliar; verify the issuer independently. Do not call a document a normal appeal case if it contains claim, court, enforcement, Order for Recovery, warrant or statutory-declaration wording.

The best next action is usually one small, reversible step: save the original, verify the issuer via an official site, view the alleged evidence, collect the documents that answer the allegation, then submit through the stated official channel before the deadline.
