# Source label: rejection_letter_checklist.md

Read a rejection letter line by line before choosing the next action. Record: operator or council name; notice reference and vehicle registration; rejection reason; date of letter and date received; appeal body; verification code or appeal link; stated payment and appeal deadlines; discount warning; and every item of evidence the letter says it considered.

Private parking: check whether the operator identifies POPLA or IAS and whether a code or route is supplied. Council PCN: check whether it is an informal-challenge reply, Notice to Owner, formal-representation response, Notice of Rejection, Charge Certificate, or Order for Recovery; these are different stages.

Build an escalation bundle from the original notice, first appeal, submission proof, rejection, all evidence, and a short list of points the response did not address. Do not miss the stated deadline while gathering more material. Court, claim, enforcement or statutory-declaration wording is not a normal appeal step: seek stage-specific help promptly.

Official route references: https://www.gov.uk/parking-tickets/challenging-a-ticket ; https://www.popla.co.uk/ ; https://theias.org/before-you-appeal
