# Source label: trade_associations_bpa_ipc.md

Many UK private parking operators belong to a trade association such as the BPA or IPC (or successor arrangements). Membership can affect which independent appeal service is offered after an operator rejection, and which code of practice expectations may be referenced.

Users should check the notice and rejection letter for the named appeal service and any code references. Trade association membership does not guarantee a successful appeal; it mainly helps identify the correct escalation route and expected process standards.
